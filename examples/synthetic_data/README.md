# Synthetic PRISMA Example Data

This directory contains a tiny, fully synthetic PRISMA input set generated by:

    python examples/generate_synthetic_example.py

It includes one synthetic DR-like GWAS file, five synthetic tissue eQTL files,
a small LD-block BED file, and a ready-to-run manifest.

These files contain no real participant-level or summary-statistics data. They
are intended only for testing installation, file formats, allele alignment, LD
block partitioning, and PRISMA output generation.

To regenerate the files deterministically:

    python examples/generate_synthetic_example.py --seed 20260515 --n_snps 240
