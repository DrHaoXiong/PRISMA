# Public Release Manifest

## Included

Core PRISMA engine:

- run_prisma.py
- loader.py
- partition.py
- builder.py
- solver.py
- tune_rank.py

Public setup files:

- README.md
- LICENSE
- NOTICE
- CITATION.cff
- requirements.txt
- .gitignore
- examples/data_manifest_template.csv
- examples/generate_synthetic_example.py
- examples/synthetic_data/: tiny fully synthetic demo input generated by examples/generate_synthetic_example.py

## Excluded Deliberately

- data/: raw GWAS, eQTL, scRNA-seq, proteomics, and metabolomics files.
- prisma_real_results/: generated manuscript results and intermediate outputs.
- archived_deprecated/ and archived_outputs_20260420/: deprecated or superseded analyses.
- scripts_pro_validation/, scripts_scrna/, scripts_plotting/, and most validation/downstream scripts: many contain local manuscript paths, real-data dependencies, or project-specific output assumptions and should be sanitized before any later public release.
- Homo_sapiens.GRCh38.110.entrez.tsv.gz: external annotation file; omit until license/source metadata are documented.

## Recommended Next Step

After reviewing this minimal public core, create a second pass for sanitized examples, synthetic test data, and manuscript-reproducibility scripts.
